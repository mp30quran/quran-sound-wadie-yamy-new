القرآن الكريم -  الشيخ وديع اليمنى 

استمع إلى تلاوات خاشعة بصوت الشيخ بجودة عالية، مع إمكانية التشغيل المتواصل والانتقال بين السور بسهولة داخل تطبيق بسيط وسريع.

الميزات:

يمكن تحميل تلاوات كاملة بصوت الشيخ
تشغيل متواصل دون انقطاع
واجهة بسيطة وسريعة
تشغيل في الخلفية مع التحكم من الإشعارات
امكانية تحميل السور الكريمة
يعمل بدون إنترنت
يحتوى على سورة الأنعام و يمكن تحميل باقى السور الكريمة
